# 设置

在[Supabase控制面板](https://app.supabase.com/)，打开你的项目，点击设置齿轮图标，然后点击 `API`。
你将需要

1. 从项目的URL中复制来自https://your-project-id.supabase.co 的项目ID
2. 从 `项目API密钥`部分复制 `匿名`的 `公共`API密钥到

将这两者粘贴到[FEZTO Supabase设置页面](https://www.fezto.xyz/#/supabase)并点击启动。
现在你可以与其他人分享和收藏包括项目ID和匿名密钥的浏览器URL，以启动相同的应用程序。
