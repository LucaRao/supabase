## Flutter

对于 `supabase-flutter`，你将在 `Supabase`类上使用静态的 `initialize()`方法。

### Flutter `initialize()`

```dart title=main.dart
Future<void> main() async {
  await Supabase.initialize(url: 'https://xyzcompany.supabase.co', anonKey: 'public-anon-key');
  runApp(MyApp());
}
```

### 访问`SupabaseClient`实例

一旦你在`main()`方法中初始化了Supabase，你就可以从你应用程序的任何地方访问`SupabaseClient`实例。

```dart
final supabase = Supabase.instance.client;
```

## 其他Dart项目

你可以使用`SupabaseClient()`方法初始化一个新的Supabase客户端。

Supabase客户端是你进入Supabase其他功能的入口是与我们在Supabase生态系统中提供的所有功能进行交互的最简单方式。

### Dart `SupabaseClient()`

```dart
final supabase = SupabaseClient('https://xyzcompany.supabase.co', 'public-anon-key');
```
