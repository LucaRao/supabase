管理API允许你以编程方式管理你的项目。

## 状态

管理API处于 `beta`状态。它在目前的状态下是可用的，但很可能会有一些突破性的变化。

## 认证

所有API请求都需要在授权头中包含一个Supabase个人令牌。`Authorization Bearer <supabase_personal_token`。
要生成或管理你的 API 令牌，请访问你的 [账户](https://app.supabase.com/account/tokens) 页面。
你的API令牌与你的用户账户有相同的权限，所以一定要保密。

```bash
$ curl https://api.supabase.com/v1/projects \
-H "Authorization: Bearer sbp_bdd0••••••••••••••••••••••••••••••••4f23"
```

所有的API请求必须经过认证并通过HTTPS进行。

## 速率限制

该API目前受制于我们的公平使用政策。在未来，我们可能会引入费率限制。
所有通过API创建的资源都受制于我们的[定价](https://supabase.com/pricing)页面上的详细定价。
