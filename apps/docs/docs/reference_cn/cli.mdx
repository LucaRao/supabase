Supabase CLI 提供了在本地开发项目并部署到 Supabase 平台的工具。
该 CLI 仍在开发中，但它包含了与您的 Supabase 项目和 Supabase 平台一起工作的所有功能。

- 在本地运行 Supabase。[`supabase start`](/docs/reference/cli/usage#supabase-start)
- 管理数据库迁移。[`supabase migration`](/docs/reference/cli/usage#supabase-migration)
- 用于发布到生产的CI/CD。[`supabase db push`](/docs/reference/cli/usage#supabase-db-push)
- 管理你的Supabase项目。[`supabase projects`](/docs/reference/cli/usage#supabase-projects)
- 直接从你的数据库模式生成类型。[`supabase gen types`](/docs/reference/cli/usage#supabase-gen)
  - 一个[社区支持的GitHub行动](https://github.com/lyqht/generate-supabase-db-types-github-action)来生成TypeScript类型
- 外壳自动完成。[`supabase completion`](/docs/reference/cli/usage#supabasecompletion)
  - 用于macOS终端的[社区支持的Fig自动完成规范](https://fig.io/manual/supabase)
